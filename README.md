## View the [Storybook](https://storybook.js.org/) Demo

### Locally

#### Install dependencies

```sh
$ nvm install
$ npm install
```

#### Start the Storybook instance

```sh
$ npm run storybook
```

#### View

Navigate to the local address shown in your terminal
